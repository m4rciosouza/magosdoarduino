# Código do curso Magos do Arduino para gravar dados simples na EEPROM Arduino

### Código de teste de EEPROM para as funções de write, update e read. Grava os valores de 1 a 10 a partir do endereço de memória 10, e depois imprime no monitor serial os valores gravados

### Artigo do projeto
[https://magosdoarduino.web.app/eeprom-arduino.html](https://magosdoarduino.web.app/eeprom-arduino.html)

### Componentes necessários
* 1x Placa Arduino UNO
